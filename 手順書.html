📂 fortunes.js の開き方

⭐ Cursor で開く場合（おすすめ）

1 Cursor を起動する
2 メニュー「ファイル」→「フォルダーを開く」を選ぶ
（またはショートカット Ctrl + Shift + O）

フォルダごと開く方法が一番ラク！一回やったら次からCursorを起動するたびに自動で開くようになるで。

🔑 1パターンの構造（必ず読んで）

鑑定パターンは以下の4つのセットで1つや：

項目説明例
title結果のタイトル（絵文字＋短い言葉）🐟 魚に呼ばれる日
body本文（関西弁・3〜5文）今日の16時43分、...
luckyラッキー情報（3個セット）ラッキー魚：サバ

実際のコードはこう書く：

// ↓ ここから1パターン
{
  title: "🍣 寿司に呼ばれる日",
  body: "今日の昼、絶対寿司が食べたくなるで。\n我慢したらアカン——それが今日の運命や。\nマグロを食べたら運気MAX。卵焼きは今日だけ天敵やから気ぃつけてや。",
  lucky: [
    { label: "ラッキーネタ",  value: "マグロ" },
    { label: "NGネタ",      value: "卵焼き（今日だけ）" },
    { label: "食べる時間",  value: "昼がベスト" },
  ]
},
// ↑ ここまで1パターン。最後の , を忘れずに！
"（ダブルクォーテーション）を消したり、{ } を消したりするとアプリが壊れるで。コピーして使うのが安全や！

➕ パターンを追加する手順

1 fortunes.js をメモ帳で開く
2 Ctrl + End でファイルの一番下に移動する
3 こういう行が見つかる： 
  ];    ← これが fortunes配列の終わり
4 その ]; の直前の行（最後の }, のあと）に新しいパターンを貼り付ける
5 Ctrl + S で保存してブラウザを開き直す 
    ...前のパターン...,   ← 前のパターンの最後
    {
      title: "🍣 寿司に呼ばれる日",
      body: "本文をここに書く。",
      lucky: [
        { label: "ラッキーアイテム", value: "マグロ" },
        { label: "NG", value: "卵焼き" },
        { label: "時間", value: "昼" },
      ]
    },                    ← ここに追加した
  ];                      ← この];は動かさない

📝 本文（body）の書き方

ルール説明
改行は \n で書く実際に画面で改行される
ダブルクォーテーション " で囲む前後に必ず " を付ける
本文中に " を使いたい場合「」 か '' を使う
長さ3〜5文がちょうどええ

例：

body: "今日の昼、絶対寿司が食べたくなるで。\n我慢したらアカン。\nマグロを食べたら今月の運気がUPするで。"

💬 煽り・ツッコミを追加する

fortunes.js の下の方に odanTexts（煽り）と tsukkomiTexts（ツッコミ）がある。

// 煽りテキスト（占い前に出るやつ）
const odanTexts = [
  'ズバリ言うで！！',
  'ヤバいやつ出たで！！',
  'ここに追加したい文を書く！！',   ← こうやって追加
];

// ツッコミテキスト（結果の下に出るやつ）
const tsukkomiTexts = [
  'ほんまかいな！！笑',
  'ここに追加したい文を書く！！',   ← こうやって追加
];
文字列は '（シングルクォーテーション）で囲んで、最後に , を付けるで！

❌ よくあるミス

ミス正しい書き方
title: 寿司に呼ばれる日
（クォーテーションなし）		 title: "🍣 寿司に呼ばれる日"
最後の , 忘れ }, で閉じる（カンマ必須）
本文に改行を直接入れた \n で改行を表現する
{ か } のどちらかが足りない 必ずペアで書く

✅ 追加できたか確認する方法

1 index.html をブラウザで開く（タブを完全に閉じてから開き直す）
2 マイクボタンを押して占いを実行する
3 追加したパターンがランダムで出てきたら成功！
パターンが100個以上あるのでランダムで出るまで何回か占ってみてや！
ブラウザが真っ白になったり動かなくなったら fortunes.js に書き間違いがある可能性があるで。AIに「確認して」と頼んだら直してもらえるで！

📤 公開手順（初回）

1 ブラウザで netlify.com/drop を開く
2 エクスプローラーで C:\Users\heart\voice-fortune フォルダを開く
3 フォルダをそのままブラウザの画面にドラッグ＆ドロップする
4 しばらく待つと https://○○○○○.netlify.app というURLが出てくる
5 そのURLをLINEやSNSで友達に送るだけ！
フォルダの中に index.html と fortunes.js が両方入っていることを確認してや！

🔄 内容を更新した時（鑑定パターンを追加したあとなど）

1 また同じように netlify.com/drop を開く
2 フォルダをドラッグ＆ドロップする
3 新しいURLが発行される（前のURLとは変わることがある）
4 新しいURLを友達に送り直す
毎回URLが変わるのが気になる場合は、Netlifyに無料アカウントを作ると同じURLを使い続けられるで。アカウント登録はメールアドレスだけでOKや！
